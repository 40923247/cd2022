期末考週

1.uram零組件繪圖過程之修改"nx12"絞鍊與"front joint零件"，使能將 "Triangular connecting rod", "left link" 與 "end link" 等零組件, 與 "right crank" 與 "right link" 的組件左右互換。

參考零件連結

2.uarm左右零件互換組裝過程youtube影片:連結

心得:因為題目組裝時要左右相反，所以要修改相關尺寸，而我使用NX12對老師的零件進行編輯尺寸使nx12絞鍊與front joint零件成左右相反的零件。 

3.uarm以 slider 控制前三軸的轉動。

心得:將組裝完成的STL檔放入coppeliaSim，然後開始進行joint的設定使三軸能旋轉，這幾天組裝過程中雖然還是有些問題，但已經有讓三軸能轉動了，目前還在努力中。

4.BG3-XML UI slider 控制功能

心得:先開coppeliaSim裡的組合檔案後按撥放鍵，然後將老師之前給的檔案使用小白窗SciTE開啟程式，然後按tool在按go就行了，就能用此程式操控UARM機械手臂的三軸運動。

5.BG3-block 方塊零件的取放模擬

心得:這題的操作方式像任務w15一樣，使用鍵盤LKJHGF控制JOINT123的轉動，吸盤的話是使用P跟Q進行吸或放的動作，操作起來還蠻有趣的 。

6.BG3-跨網路 Python remote API

心得:查詢各組員的IVP4位置，然後修改程式為組員的IVP4位置，儲存後按TOOLS在按GO就可以開始遠端操作UARM機械手臂的三軸運動。
