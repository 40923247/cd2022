設法以隨身程式系統編譯此程式碼:
1.開啟隨身程式系統進到TMP>CD2022中git clone https://github.com/mdecourse/scarysim.git
2.開啟白窗修改 scarysim.pro, 蓋掉第 11 行, 成為 # LIBS += -lglut -lGLU
3.cd scarysim
4.qmake -o Makefile scarysim.pro
5.mingw32-make -f Makefile.Release 可以在 release 目錄取得 scarysim.ex
6.因為 scarysim.exe 執行需要 base.obj, arm1.obj, arm2.obj, arm3.obj 等零件外型檔案, 以及對應的 .mtl 材質檔案, 因此可將 scarysim.exe 移至倉儲根目錄, 並且需要 msys64_20210419\mingw64\bin 路徑中的 dll 動態連結程式庫配合執行.
7.複製simExtMtb.dll檔開啟coppeliasim410_msys2資料夾並將舊的simExtMtb.dll改成simExtMtbold.dll再貼上simExtMtb.dll檔即可完成。
